## 🚀 How to Run a Project

1. Navigate to the project folder:

```bash
cd l1-beginner/project-1-simple-rest-api
```

2. Build Docker image:

```bash
docker build -t your-image-name .
```

3. Run container:

```bash
docker run -p 8080:8080 your-image-name
```
